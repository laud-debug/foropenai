# Profil fournisseur — Games Workshop

## Références
- La référence interne Rebel Forge attendue pour les imports structurés est `GW-` suivi du Product Code lorsque cette règle est applicable.
- Le Product Code source doit être conservé séparément pour permettre les rapprochements.
- Ne pas déduire une identité produit à partir d'un nom proche lorsque le Product Code diffère.

## Catégorisation
La catégorie par défaut des imports de précommande peut être « Réservation GW », mais elle doit rester modifiable avec recherche dans l'arbre de catégories.

## Données attendues
Release Date, Module, System, Race, SS Code, Product Code, Order, Description, RSL, Qty, Barcode, Commodity, Country, Weight, Cube, FRR, EUD, CHR et CHD lorsqu'elles existent dans le fichier fournisseur.

## Publication
Le choix en ligne / hors ligne est obligatoire. Les précommandes doivent être identifiables comme telles et ne pas être publiées par défaut lorsqu'une donnée essentielle manque.
