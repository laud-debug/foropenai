# Pack Codex — RF Product Factory

Ce dossier est prêt à être copié à la racine du module `rfproductfactory`.

## Installation dans le dépôt
Copier :
- `AGENTS.md` à la racine de `rfproductfactory` ;
- le dossier `docs/` à la racine de `rfproductfactory` ;
- le dossier `codex-missions/` à la racine de `rfproductfactory`.

Puis lancer dans VS Code :

```bash
git checkout -b docs/PF-001-codex-foundation
git add AGENTS.md docs codex-missions
git commit -m "Add Product Factory Codex documentation framework"
git push -u origin docs/PF-001-codex-foundation
```

## Première mission
Copier intégralement le contenu de :

`codex-missions/MISSION-01-CONSOLIDATION.md`

et le donner à Codex depuis la racine du dépôt.

Codex doit d'abord consolider la documentation. Il ne doit pas modifier le PHP lors de cette première mission.
