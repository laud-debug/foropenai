# UX — Tableau de bord

Le dashboard doit permettre de comprendre la situation en moins de trente secondes.

## Indicateurs prioritaires
- Analyses en attente.
- Produits à valider.
- Créations du jour.
- Enrichissements du jour.
- Erreurs bloquantes.
- Jobs interrompus.
- Nouveaux produits fournisseurs détectés.
- Temps économisé estimé.

## Listes opérationnelles
- Derniers jobs avec statut, source, nombre de produits et durée.
- Produits nécessitant une décision humaine.
- Erreurs récentes avec lien direct vers le job.
- Raccourcis : nouvelle analyse, import fichier, reprendre un job.

## Règles
Chaque KPI ouvre la liste filtrée correspondante. Aucun indicateur décoratif sans action utile.
